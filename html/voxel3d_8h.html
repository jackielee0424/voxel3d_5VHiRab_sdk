◆ voxel3d_dev_fw_upgrade()

VOXEL3D_API_DLL int voxel3d_dev_fw_upgrade ( char * dev_sn,
char * file_path,
unsigned char(* fw_upgrade_cb )(int state, unsigned int percent_complete) )

5voxel 5VHiRab device firmware upgrade utility

Warning
Device firmware upgrade utility allows both upgrade to new version of firmware and also downgrade to older version. User can use voxel3d_read_fw_version() to confirm the firmware version running on device. It can also be used to confirm if the firmware version on device after upgrade.
Parameters
[in]dev_sndevice S/N. Input S/N with NULL pointer or empty string will initialize the 1st scanned device
[in]file_pathpoint to the location of the new firmware image string
Returns
true: completed sending specific firmware to device for upgrade successfully
< 0: upgrade failure

Function to poll the state and 5voxel 5VHiRab device firmware upgrade utility.

Parameters
[in]dev_sndevice S/N. Input S/N with NULL pointer or empty string will initialize the 1st scanned device
[out]statereference of the state varaible for API to write current fw download state state: < 0 (error), 0: (initial), 1 (downloading), 2 (complete)
[out]percent_completereference of the percentage varaible for API to write current percentage of completion percent_complete: 0 ~ 100
Returns
true: In upgrade procedure
< 0: Not in upgrade procedure Note: while output < 0, state & percent_complete can be used to know if the previous upgrade had error or completed withtout failure

VOXEL3D_API_DLL int voxel3d_tof_set_conf_threshold ( char * dev_sn,

Set run-time ToF confidence threshold to 5voxel 5VHiRab device.

Warning
  1. Call this function after voxel3d_tof_init() is completed and successfully, otherwise, it returns false
  2. Corresponding pixel depth will be zero if its confidence is lower than threshold
  3. Configured threshold will go back to default when camera is power-cycled
Parameters
[in]dev_sndevice S/N. Input S/N with NULL pointer or empty string will initialize the 1st scanned device
[in]conf_threshold0~4095
Returns
true: set confidence threshold successfully
< 0: failed to set confidence threshold
